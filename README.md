# Project setup
# Client
```
cd client
npm install
npm install vue-router
npm install axios vue-axios
npm install --save vue-apexcharts
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

# Server
```
docker-compose up
```
